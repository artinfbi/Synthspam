# Synthspam
Effective perisan sms bomber tool


## Installation



```bash
  sudo apt-get install tor

  git clone https://github.com/artinfbi/Synthspam

  cd Synthspam

  pip install -r requirements.txt
  
```
## run 
```bash
    tor HTTPTunnelPort 8000
    python3 Synthspam.py
```
